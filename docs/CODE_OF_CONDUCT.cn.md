# Contributor Covenant 行为准则

[English](../CODE_OF_CONDUCT.md) | 中文简体

## 我们的承诺

作为成员、贡献者和领导者，我们承诺使参与本社区的每个人都不受骚扰，不论年龄、体型、可见或不可见的残疾、族裔、性征、性别认同与表达、经验水平、教育、社会经济地位、国籍、外貌、种族、种姓、肤色、宗教或性取向。

我们承诺以有助于营造开放、友好、多元、包容和健康社区的方式行事和互动。

## 我们的标准

有助于为本社区营造积极环境的行为示例包括：

* 对他人表现出同理心和善意
* 尊重不同的观点、看法和经验
* 给予并优雅地接受建设性反馈
* 承担责任，向受影响者道歉，并从经验中学习
* 关注不仅对我们个人最好，而且对整个社区最好的事情

不可接受的行为示例包括：

* 使用性化的语言或意象，以及任何形式的性关注或性接近
* 挑衅、侮辱或贬损性评论，以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可发布他人私人信息（如物理地址或电子邮箱）
* 其他在专业环境中可被合理视为不当的行为

## 执行责任

社区领导者有责任澄清和执行可接受行为的标准，并将对任何他们认为不当、威胁、冒犯或有害的行为采取适当和公正的纠正措施。

社区领导者有权且有责任删除、编辑或拒绝与行为准则不符的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，并在适当时传达 moderation 决定的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间中正式代表社区的情况。代表社区的示例包括使用官方电子邮件地址、通过官方社交媒体账号发帖，或作为在线或线下活动的指定代表。

## 执行

对滥用、骚扰或其他不可接受行为的举报，可发送至负责执行的社区领导者：[在 GitHub 上开启 Issue](https://github.com/OpenCML/Camel/issues)（如涉及敏感内容，可使用 GitHub 的私有安全通告功能）。所有投诉都将得到及时和公正的审查与调查。

所有社区领导者都有义务尊重任何事件举报人的隐私和安全。

## 执行指南

社区领导者将遵循以下社区影响指南，来确定其认为违反本行为准则的任何行为的后果：

### 1. 纠正

**社区影响**：使用不当语言或其他在社区中被认为不专业或不受欢迎的行为。

**后果**：社区领导者的私下书面警告，说明违规性质及行为不当的原因。可要求公开道歉。

### 2. 警告

**社区影响**：单次或一系列违规行为。

**后果**：附带持续行为后果的警告。在指定时间内不得与相关人员互动，包括与执行行为准则的人员主动互动。这包括避免在社区空间以及社交媒体等外部渠道的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内临时禁止与社区进行任何形式的互动或公开交流。在此期间不得与相关人员公开或私下互动，包括与执行行为准则的人员主动互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、对个人的骚扰，或对某类人的攻击或贬损。

**后果**：永久禁止在社区内进行任何形式的公开互动。
